Slide 01 // Business Concept & Traffic Engine

從線上情感迷因到實體3D記憶公仔

核心商業想法 (O2O)

打破傳統3D打印店缺乏前端流量與高昂建模成本的瓶頸。利用低門檻的 AI 趣味生成網頁吸引跨世代人群上傳照片，透過 AI 算法一鍵轉化為 3D 模型，最終引流至實體地舖進行「3D打印設備租賃體驗」，實現高溢價的情感消費閉環。

多元線上引流場景（低門檻、高傳播）

  • 毛孩經濟：上傳寵物照，生成皮克斯風或超級英雄公仔。
  • 全家福紀念：打破傳統平面相片，將家族合照立體化為常駐客廳的擺設。
  • 情侶與婚慶：客製化 Q 版情侶公仔、搞怪迷因互動模型，主打節慶送禮。
  • 畢業與里程碑：上傳畢業照，將真人面孔與畢業熊、學士服AI完美融合打印。

Slide 02 // Technical Architecture & Lean Validation

技術架構架設與精實商模驗證

技術模組 A

AI 圖像生成與預覽

  • 2D 風格化：雲端部署 Stable Diffusion + LoRA 微調模型，快速將人臉/寵物轉為特定主題風格。
  • 單圖 3D 重建：串接 Tripo3D / Meshy API，或自主部署 InstantMesh，在 30 秒內將 2D 圖片輸出為帶紋理的 `.gltf` 模型。
技術模組 B

數據自動化與硬體

  • 模型自動自動化修復：調用 Blender Python API 自動進行網格閉合（Make Solid）與一鍵加厚，確保模型符合打印規格。
  • 全彩多色硬體：店面部署拓竹（Bambulab）X1C + AMS 自動色彩系統，降低消費者上手門檻。
精實創業驗證

三階段市場驗證法

  • 第一階段：線上接單（全輕資產）：上線網頁收集照片，在家用1台機器印製並郵寄，驗證客群付費意願與轉化率。
  • 第二階段：文創快閃（體驗驗證）：進駐市集或寵物友善咖啡廳，測試現場等待經濟與機台租賃計費模式。
  • 第三階段：地舖落地（規模化）：鎖定高坪效商圈，建立 AI 創客與情感文創空間。
